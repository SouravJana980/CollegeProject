# 🌬️ AirQ - Realtime Air Quality Dashboard

AirQ is a modern realtime air quality monitoring dashboard built using HTML, CSS, JavaScript, Firebase, ESP32, and WeatherAPI.

---

# 🚀 Features

- Realtime ESP32 sensor monitoring
- WeatherAPI integration
- PM2.5, AQI, CO, Temperature & Humidity tracking
- Interactive realtime charts using Chart.js
- Firebase Realtime Database integration
- Modern glassmorphism UI
- Responsive design
- Event-driven architecture

---

# 🛠️ Tech Stack

## Frontend
- HTML5
- CSS3
- JavaScript (ES Modules)

## Libraries
- Chart.js
- Lucide Icons

## Backend / Cloud
- Firebase Realtime Database
- WeatherAPI

## Hardware
- ESP32
- MQ135
- MQ7
- MQ2
- DHT11/DHT22
